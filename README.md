Daspi JupyterLite provides an in-browser interactive environment to explore data analysis, statistics, and process improvement workflows using the Daspi package. This GitHub Pages site hosts ready-to-run JupyterLite notebooks and examples so users can try Daspi features, follow guided tutorials, and experiment with sample datasets — all without installing anything locally.

Included are demonstration notebooks for common tasks (data cleaning, statistical summaries, visualization, process capability analysis, and improvement suggestions), a quickstart notebook to get started, and configuration to run Daspi in the JupyterLite environment. Ideal for demos, teaching, and quick experimentation.
